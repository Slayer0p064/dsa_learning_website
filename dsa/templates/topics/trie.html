Trie (Prefix Tree) in Data Structures

A Trie is a tree-like data structure used for storing strings for efficient prefix matching.

What is a Trie?

A Trie is used for fast retrieval of strings with common prefixes.

  • Each node represents a character.
  • Used in autocomplete and spell-checking systems.

Common Trie Operations

  1. Insertion.
  2. Search.
  3. Prefix Matching.

Python Code Examples


# Trie Node
class TrieNode:
    def __init__(self):
        self.children = {}
        self.is_end_of_word = False

# Trie Implementation
class Trie:
    def __init__(self):
        self.root = TrieNode()

    def insert(self, word):
        node = self.root
        for char in word:
            if char not in node.children:
                node.children[char] = TrieNode()
            node = node.children[char]
        node.is_end_of_word = True

    def search(self, word):
        node = self.root
        for char in word:
            if char not in node.children:
                return False
            node = node.children[char]
        return node.is_end_of_word

trie = Trie()
trie.insert("apple")
print(trie.search("apple"))  # Output: True
