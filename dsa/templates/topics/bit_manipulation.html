Bit Manipulation in Data Structures

Bit manipulation involves performing operations directly on binary representations of numbers, often leading to optimized solutions.

What is Bit Manipulation?

Bit manipulation is a technique used in programming to perform operations at the bit level. It is commonly used in scenarios requiring high efficiency, such as competitive programming and system-level optimizations.

  • Bitwise AND (&) - Used to clear bits or check if a number is even.
  • Bitwise OR (|) - Used to set bits.
  • Bitwise XOR (^) - Used for toggling bits and finding unique elements.
  • Bitwise NOT (~) - Used for inverting bits.
  • Left Shift (<<) - Used for fast multiplication by powers of 2.
  • Right Shift (>>) - Used for fast division by powers of 2.

Common Bit Manipulation Techniques

  1. Checking if a number is even or odd: Using the least significant bit.
  2. Swapping two numbers without a temporary variable: Using XOR.
  3. Finding the number of set bits (Hamming Weight): Counting the 1s in binary representation.
  4. Checking if a number is a power of 2: A number is a power of 2 if it has exactly one set bit.
  5. Finding the rightmost set bit: Useful for bit masking.

Python Code Examples


# Check if a number is even or odd
def is_even(n):
    return (n & 1) == 0

print(is_even(4))  # Output: True
print(is_even(5))  # Output: False

# Swapping two numbers without a temp variable
def swap(a, b):
    a = a ^ b
    b = a ^ b
    a = a ^ b
    return a, b

print(swap(3, 5))  # Output: (5, 3)

# Count the number of set bits (Hamming Weight)
def count_set_bits(n):
    count = 0
    while n:
        count += n & 1
        n >>= 1
    return count

print(count_set_bits(9))  # Output: 2 (binary: 1001)

# Check if a number is a power of 2
def is_power_of_two(n):
    return n > 0 and (n & (n - 1)) == 0

print(is_power_of_two(8))  # Output: True
print(is_power_of_two(10)) # Output: False

Applications of Bit Manipulation

  • Optimizing algorithms for competitive programming.
  • Data compression and encryption.
  • Graphics rendering (e.g., bitwise operations in shaders).
  • Networking (e.g., setting or clearing flags in packets).
  • Efficient mathematical computations in embedded systems.
